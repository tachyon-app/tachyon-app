# Tachyon Documentation

Welcome to the Tachyon documentation! This directory contains detailed guides for all features.

## Feature Documentation

### Core Features

- **[Calculator](CALCULATOR.md)** - Math expressions, unit conversions, and currency conversion
  - Basic arithmetic and advanced functions
  - Unit conversions (length, temperature, mass, volume, time, data)
  - Live currency conversion with 30+ currencies

- **[Date & Time](DATE_TIME.md)** - Natural language dates and time calculations
  - Unix timestamp conversion (seconds, milliseconds, microseconds)
  - Natural language parsing (today, tomorrow, next friday, etc.)
  - Date arithmetic (today + 3 days, now - 2 hours)
  - Timezone conversions (time in tokyo, 5pm london in nyc)
  - Duration calculations (days until christmas)
  - Week & day numbers

- **[System Commands](SYSTEM_COMMANDS.md)** - 26 macOS system controls
  - Power & session management (sleep, lock, restart, shutdown)
  - System settings (dark mode, Wi-Fi, Bluetooth, Night Shift)
  - Audio controls (mute, volume levels)
  - File management (empty trash, hidden files, eject disks)
  - App management (quit all, hide all, show desktop)

- **[Focus Mode](FOCUS_MODE.md)** - Productivity timer with music integration
  - Timer sessions (15, 25, 45, 60 minutes or custom)
  - Spotify music integration (paste URLs to add focus music)
  - Glowing screen border (customizable color and thickness)
  - Floating timer bar (draggable, minimizable)

### Productivity Features

- **[Search](SEARCH.md)** - Global search and app launcher
  - Fuzzy matching for apps
  - Quick launch with keyboard

- **[Quick Links](QUICKLINKS.md)** - Custom URL templates
  - Create shortcuts to frequently visited URLs
  - Use placeholders for dynamic content

- **[Search Engines](SEARCH_ENGINES.md)** - Custom search templates
  - Add your favorite search engines
  - Quick web searches from the launcher

- **[Window Management](WINDOW_MANAGEMENT.md)** - Rectangle-style window snapping
  - Keyboard shortcuts for window positioning
  - Customizable hotkeys
  - Multi-monitor support

- **[Scripts](SCRIPTS.md)** - Raycast-compatible script execution
  - Run custom scripts from the launcher
  - Support for multiple script modes
  - Package management

### Settings & Configuration

- **[Settings](SETTINGS.md)** - Application settings and preferences
  - Global hotkey configuration
  - Plugin management
  - Appearance customization

## Quick Start

1. **Launch Tachyon**: Press `Cmd+Space` (or your configured hotkey)
2. **Type your query**: Start typing to search apps, calculate, or execute commands
3. **Press Enter**: Execute the selected action or copy the result

## Tips & Tricks

- **Case Insensitive**: All queries work regardless of capitalization
- **Fuzzy Matching**: Type partial words to find what you need
- **Copy Results**: Press Enter to copy calculator and date results
- **Quick Access**: Use keyboard shortcuts for instant access
- **Multiple Formats**: Date results include Unix, ISO, RFC, and human-readable formats
