# Settings

Tachyon is highly customizable. You can access the settings by pressing `Cmd + ,` while the Tachyon window is open, or by searching for "Settings" within Tachyon itself.

## General
- **Launch at Login**: Toggle whether Tachyon starts automatically when you log in.
- **Hotkey**: Change the global keyboard shortcut to toggle Tachyon window (Default: `Option + Space`).
- **Theme**: Choose between Light, Dark, or System theme.
